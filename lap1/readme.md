# Project Setup Documentation

## Creating My GitHub Account
1. I went to https://github.com
2. Clicked on "Sign up"
3. Entered my email address, username, and password
4. Verified my email
5. Completed the account setup

## Creating a New Repository
1. After logging in, I clicked on the "+" icon at the top right
2. Selected "New repository"
3. Entered a repository name
4. Chose public visibility
5. Checked "Add a README file"
6. Clicked "Create repository"

## Creating the README File
- GitHub automatically created the README.md file when I selected the option.
- I edited the README file by clicking the pencil (edit) icon.
- Wrote the project documentation.
- Clicked "Commit changes" to save.
